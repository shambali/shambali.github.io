Okular

Okular, PDF, çizgi roman, EPub, görsel, Markdown ve daha birçok belgeyi okuyup görüntüleyebilen çok yönlü bir uygulamadır.



Tek komutla indirin! (Flatpak)

flatpak install flathub org.kde.okular
WinGet ile indirin!

winget install --id=KDE.Okular -e
